<dl>,<dt>,<dd>

  • Описание:

    • Тройка элементов предназначена для создания списка определений. Каждый такой список начинается с контейнера <dl>, куда входит тег <dt> создающий термин и тег <dd> задающий определение этого термина.

  • Синтаксис:

    • <dl>
          <dt>Термин 1</dt>
          <dd>Определение 1</dd>
          <dt>Термин 2</dt>
          <dt>Определение 2</dt>
    </dl>

  • Атрибуты:

    • Не доступны.

  • Закрывающий тег:

    • Закрывающий тег </dd> не обязателен, поскольку следующий тег сообщает о завершении предыдущего элемента.
